# 種目詳細ページ設計書

## 概要
診断結果画面に表示される各種目名をクリックすると、その種目の詳細情報を表示する専用ページに遷移する機能を追加する。

---

## 機能要件

### 1. 診断結果画面の変更
- 結果画面に表示される各種目名（例：ベンチプレス、スクワット等）をクリック可能なリンクにする
- クリック時に種目詳細ページへ遷移する
- 種目名にはホバー時のスタイル変更を追加し、クリック可能であることを視覚的に示す

### 2. 種目詳細ページ（exercise.html）

#### ページ構成（上から順に）

```
┌─────────────────────────────────────┐
│           ヘッダー                    │
│    ロゴ / 戻るボタン                  │
├─────────────────────────────────────┤
│                                     │
│         種目名（タイトル）             │
│         対象部位バッジ                │
│                                     │
├─────────────────────────────────────┤
│                                     │
│     ┌───────────────────────┐       │
│     │                       │       │
│     │    動画エリア          │       │
│     │   （YouTube埋め込み）   │       │
│     │                       │       │
│     └───────────────────────┘       │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  【意識すること】                     │
│   ・ポイント1                        │
│   ・ポイント2                        │
│   ・ポイント3                        │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  【気を付けること】                   │
│   ・注意点1                          │
│   ・注意点2                          │
│   ・注意点3                          │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  【代替種目】（任意）                  │
│   ・代替種目リスト                    │
│                                     │
└─────────────────────────────────────┘
```

#### 表示項目

| 項目 | 説明 |
|------|------|
| 種目名 | 種目の正式名称（例：ベンチプレス） |
| 対象部位 | 鍛えられる部位（例：胸、三頭筋） |
| 動画 | YouTube動画の埋め込み（16:9比率） |
| 意識すること | 効果を高めるためのフォームポイント（3〜5項目） |
| 気を付けること | 怪我防止のための注意点（2〜4項目） |
| 代替種目 | 同じ部位を鍛える別の種目（任意表示） |

## 画面遷移

```
index.html（診断）
    ↓ 診断完了
結果画面（種目一覧表示）
    ↓ 種目名クリック
exercise.html?id=bench-press（種目詳細）
    ↓ 戻るボタン
結果画面に戻る
```

## UI/UXデザイン

### 種目名リンクのスタイル
```css
.exercise-link {
  color: #f5576c;
  text-decoration: underline;
  cursor: pointer;
  transition: all 0.2s ease;
}

.exercise-link:hover {
  color: #4facfe;
  text-decoration-thickness: 2px;
}
```

### 動画エリア
- アスペクト比 16:9 を維持
- 最大幅 100%、レスポンシブ対応
- 角丸（border-radius: 16px）
- ドロップシャドウで浮き上がり感

### セクションカード
- 「意識すること」「気を付けること」は別カードで表示
- アイコンで視覚的に区別
  - 意識すること → チェックマークアイコン
  - 気を付けること → 注意アイコン

## 技術仕様

### ファイル構成
```
/
├── index.html          # 診断ページ（既存・改修）
├── exercise.html       # 種目詳細ページ（新規）
├── js/
│   └── exercises.js    # 種目データ（新規）
└── doc/
    └── 種目詳細ページ設計書.md
```

### URLパラメータ
- `exercise.html?id={種目ID}`
- 例：`exercise.html?id=bench-press`

### 戻る機能
- ブラウザの履歴を使用（`history.back()`）
- または診断結果を保持して戻る

## 動画について

### 方針
- **YouTube上の既存フィットネス動画**を埋め込みで使用する
- YouTube埋め込み機能は規約上許可されている

### 動画選定基準
| 基準 | 詳細 |
|------|------|
| 内容 | 正しいフォームを分かりやすく解説している |
| 長さ | 3〜10分程度（短すぎず長すぎない） |
| 画質 | 720p以上で見やすい |
| 言語 | 日本語解説が望ましい |
| 信頼性 | 登録者数が多い、専門家が解説している |

### 参考チャンネル候補
- なかやまきんに君
- RIZAP公式
- ザ・きんにくTV
- 山本義徳 筋トレ大学
- その他フィットネス専門チャンネル

### 埋め込み方法
```html
<iframe
  src="https://www.youtube.com/embed/{動画ID}"
  frameborder="0"
  allowfullscreen>
</iframe>
```
